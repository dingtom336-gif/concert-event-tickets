# concert-event-tickets

Find concerts, live performances, sports events, and festivals. Get ticket prices, seating info, and direct booking links.

## Overview

Find concerts, live performances, sports events, and festivals. Get ticket prices, seating info, and direct booking links. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install concert-event-tickets
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/concert-event-tickets
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Find concerts, live performances, sports events, and festivals. Get ticket prices, seating info, and direct booking links.

**Command type:** Single-command
